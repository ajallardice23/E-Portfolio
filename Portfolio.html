Zombie Game (Bangor University assignment)

Module Grade: 86%

During my third year at University I took an Advanced Game Development Module. The main assignment of the module was to create a game using a game engine of my choice, due to having previous experience with Unity, I found this to be the best option.


I decided to create a 3d game similar to Call of Duty Zombies in a low poly style. I created all the environment and assets myself mainly using blender.


Main Features

Round system

In my game, I set up a game controller which releases waves of zombies, making more zombies spawn per wave completed.

Reload System

I implemented a reload system in two different guns with different reload times.

Particle Systems

I used the built-in Unity particle system to create effects such as sparks, water and fog.

Coin System

I created a coin system allowing users to upgrade the gun when you have enough coins. The coins have a chance to spawn per kill.

Power up System

I created some power ups that can spawn when a zombie is killed. These include gliders, more ammo, coins.

Map Level Up

The user is able to buy certain areas with their coins to extend the play space.

Experimental Joints

I experimented with joints for the first time and was able to make seesaws out of planks I created using the players gravity.

MiniMap

I added a seperate camera that shows a birds eye view when a hotkey is pressed. Resembles a game minimap.

Runtyper (GameJam)

This game was created when I took part in my first ever game jam. The theme for the game jam was "run".


After seeing the theme, I started creating a typing game where you race against time, creating all the assets myself.


Main Features

Random Letter Selection

For the player to move forward, they have to match the random letter displayed on their screen picked from an array.

Difficulty

On the game menu, a difficulty selection appears where players can choose what skill level AI they want to play against.

AI System

The opponents run against the player using an AI system. The player has to type faster than the AI system can run.

Placement System

Once the player passes the finish line, they are shown what place they came in the race.

Win Count and Streak

Tracks players wins and wins in a row

Coin system and shop

For each win the player receives a coin. Coins can be saved up and spent on cosmetics.

Individual Project and Dissertation (Bangor University assignment)

Module Grade: 74%

For my final University project I decided to focus on renewable energy because I am all for a greener future.


I created a web-based tool using html, css, javascript and qgis which determines offshore wind turbine output (MWh/year) from a selected place around the UK in an attempt to prove that floating offshore turbines are much more efficient than fixed offshore turbines and other current commonly used methods.
