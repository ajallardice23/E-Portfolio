Zombie Game (Bangor University assignment)

Module Grade: 86%

During my third year at University I took an Advanced Game Development Module. The main assignment of the module was to create a game using a game engine of my choice, due to having previous experience with Unity, I found this to be the best option.


I decided to create a 3d game similar to Call of Duty Zombies in a low poly style. I created all the environment and assets myself mainly using blender.


Main Features

Round system

In my game, I set up a game controller which releases waves of zombies, making more zombies spawn per wave completed.

Reload System

I implemented a reload system in two different guns with different reload times.

Particle Systems

I used the built-in Unity particle system to create effects such as sparks, water and fog.

Coin System

I created a coin system allowing users to upgrade the gun when you have enough coins. The coins have a chance to spawn per kill.

Power up System

I created some power ups that can spawn when a zombie is killed. These include gliders, more ammo, coins.

Map Level Up

The user is able to buy certain areas with their coins to extend the play space.

Experimental Joints

I experimented with joints for the first time and was able to make seesaws with the players gravity.

MiniMap

I added a separate camera that shows a birds eye view when a hotkey is pressed. Resembles a game minimap.
